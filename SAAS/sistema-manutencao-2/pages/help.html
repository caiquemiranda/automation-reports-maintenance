Perguntas Frequentes

Como criar uma nova Ordem de Serviço?

Para criar uma nova Ordem de Serviço, acesse o menu "OS" na barra lateral e selecione o tipo de OS que deseja criar (Corretiva, Planejada ou Preventiva). Preencha todos os campos obrigatórios no formulário e clique em "Salvar". A nova OS será registrada no sistema e aparecerá na lista de ordens de serviço.

Como gerar relatórios mensais?

Para gerar relatórios mensais, acesse a página "Report Mensal" no menu lateral. Selecione o mês e o ano desejados nos filtros disponíveis e clique em "Gerar Relatório". O sistema processará os dados e exibirá os gráficos e tabelas com as informações solicitadas. Você pode exportar o relatório em diferentes formatos clicando no botão "Exportar".

Como programar uma manutenção preventiva?

Para programar uma manutenção preventiva, acesse a página "Cronograma Mensal" no menu lateral. Clique no botão "Nova Programação" e selecione o equipamento, o técnico responsável e a data desejada. Defina o tipo de manutenção como "Preventiva" e adicione informações adicionais se necessário. Salve a programação e ela aparecerá no calendário. Alternativamente, você pode criar uma OS Preventiva diretamente pela página correspondente.

Como encontrar um dispositivo específico no mapa?

Para encontrar um dispositivo específico no mapa, acesse a página "Mapa de Dispositivos" e use o campo de busca localizado acima do mapa. Digite o nome ou ID do dispositivo e o sistema irá localizá-lo automaticamente. Você também pode filtrar os dispositivos por tipo, status ou localização usando os filtros disponíveis na página.

Como exportar dados do sistema?

Para exportar dados do sistema, acesse a página "Dados > Exportação" no menu lateral. Selecione o tipo de dados que deseja exportar (OS, Equipamentos, Relatórios, etc.), defina o período e o formato desejado (CSV, Excel, PDF) e clique em "Exportar". O sistema processará a solicitação e disponibilizará o arquivo para download quando estiver pronto.
